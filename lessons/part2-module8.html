Part 2, Module 8: Data Processing and Serialization

Working with JSON: Encoding and Decoding

Go provides built-in support for JSON encoding and decoding, including to and from built-in and custom data types.

Encoding (Marshalling)
type Response1 struct {
    Page   int
    Fruits []string
}
res1D := &Response1{
    Page:   1,
    Fruits: []string{"apple", "peach", "pear"}}
res1B, _ := json.Marshal(res1D)
fmt.Println(string(res1B))
Decoding (Unmarshalling)
str := `{"page": 1, "fruits": ["apple", "peach"]}`
res := Response2{}
json.Unmarshal([]byte(str), &res)
fmt.Println(res.Fruits[0])
Processing other common formats like CSV and XML

Go's standard library also includes packages for working with other common data formats like CSV and XML.

Handling data streams and transformations

The io package provides the io.Reader and io.Writer interfaces, which are used throughout the standard library to represent data streams.
