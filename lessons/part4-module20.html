Part 4, Module 20: Capstone: User & Auth API

Build the REST API for user accounts (register, login)

In this module, you will build the REST API for user registration and login. This will involve creating the necessary routes, handlers, and database queries.

Use Fiber, `sqlc`, and your JWT implementation

You will use the tools and techniques you've learned throughout this course, including:

  • Fiber: For building the web server and handling requests.
  • `sqlc`: For generating type-safe Go code for your database queries.
  • JWT: For implementing authentication.
Secure the necessary endpoints

You will need to secure certain endpoints (e.g., user profiles) using your JWT authentication middleware.

Assignment

Your assignment for this module is to:

  1. Implement the user registration endpoint.
  2. Implement the user login endpoint.
  3. Create a protected endpoint that can only be accessed by authenticated users.
