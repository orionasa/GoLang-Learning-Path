Part 2, Module 7: File System I/O

Reading files using the `os` and `io` packages

The os package provides functions for interacting with the operating system, including file I/O.

dat, err := os.ReadFile("/tmp/dat")
check(err)
fmt.Print(string(dat))
Writing data to files

You can write data to a file using the os.WriteFile function.

d1 := []byte("hello\ngo\n")
err := os.WriteFile("/tmp/dat1", d1, 0644)
check(err)
Buffering I/O for performance with `bufio`

The bufio package implements buffered I/O. It's useful for improving performance when working with large files.

f, err := os.Create("/tmp/dat2")
check(err)
defer f.Close()

w := bufio.NewWriter(f)
n4, err := w.WriteString("buffered\n")
check(err)
fmt.Printf("wrote %d bytes\n", n4)
w.Flush()
Navigating the file system

The os package also provides functions for navigating the file system, such as reading directories.

files, err := os.ReadDir(".")
if err != nil {
    log.Fatal(err)
}

for _, file := range files {
    fmt.Println(file.Name())
}
