Part 1, Module 2: Basic Syntax and Data Types

Packages and Imports

Every Go program is made up of packages. The main package is the entry point for executable programs.

You can import other packages using the import keyword.

package main

import (
    "fmt"
    "math"
)

func main() {
    fmt.Println("The square root of 4 is", math.Sqrt(4))
}
Variables, Constants, and Zero Values

Variables can be declared using the var keyword. Go is statically typed, so you must specify the type.

var age int = 30
var name = "John" // Type inference
city := "New York" // Short variable declaration

Constants are declared using the const keyword and cannot be changed.

const pi = 3.14159

Variables declared without an explicit initial value are given their zero value (e.g., 0 for numeric types, false for booleans, and "" for strings).

Basic Data Types

Go has several built-in data types:

  • int: Integer type (e.g., int8, int16, int32, int64).
  • float64: Floating-point numbers.
  • string: A sequence of bytes.
  • bool: true or false.
Control Flow

Go provides several control flow statements:

If/Else
if age >= 18 {
    fmt.Println("You are an adult.")
} else {
    fmt.Println("You are a minor.")
}
For Loops

Go has only one looping construct, the for loop.

// Standard for loop
for i := 0; i < 5; i++ {
    fmt.Println(i)
}

// While loop style
sum := 1
for sum < 1000 {
    sum += sum
}
Switch
switch os := runtime.GOOS; os {
case "darwin":
    fmt.Println("OS X.")
case "linux":
    fmt.Println("Linux.")
default:
    fmt.Printf("%s.\n", os)
}
