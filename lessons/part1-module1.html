Part 1, Module 1: Introduction & Setup

Why Go?

Go, also known as Golang, is a statically typed, compiled programming language designed at Google. It is known for its simplicity, efficiency, and strong support for concurrent programming.

  • Concurrency: Go's goroutines and channels make it easy to write highly concurrent and parallel programs.
  • Performance: Go is a compiled language that compiles down to machine code, resulting in excellent performance.
  • Simplicity: The language has a small and simple syntax, making it easy to learn and read.
Installing and Configuring the Go Toolchain

Before you can start writing Go code, you need to install the Go toolchain.

  1. Download the installer for your operating system from the official Go website.
  2. Follow the installation instructions for your OS.
  3. Verify the installation by opening a terminal and running: 
    go version
  4. GOPATH and GOMODULES: Go uses a workspace to manage source code files. By default, your workspace is located at $HOME/go. Go 1.11 introduced modules, which is the standard way to manage dependencies. Ensure you have GO111MODULE=on in your environment variables.
Your First Program: "Hello, World!"

Let's write a simple "Hello, World!" program in Go.

package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
Using the Go CLI

The Go command-line interface is a powerful tool for working with Go code.

  • go run <filename.go>: Compiles and runs the specified Go file.
  • go build <filename.go>: Compiles the Go file and creates an executable file.
  • go fmt <filename.go>: Formats the Go code according to Go's style guidelines.
Recommended IDEs and Setup

While you can use any text editor to write Go code, using an IDE with Go support can significantly improve your productivity.

  • Visual Studio Code: A popular choice with an excellent Go extension that provides features like IntelliSense, code navigation, and debugging.
  • GoLand: A full-featured IDE for Go development from JetBrains.
